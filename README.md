🚀 LoyaltyChain — Cross-Brand Loyalty Points Exchange
📌 Overview

LoyaltyChain is a blockchain-based platform that enables users to earn, swap, and redeem loyalty points across multiple brands.

💡 Think Paytm coins, Amazon cashback, Swiggy credits — what if you could use them interchangeably?

This project solves the problem of isolated loyalty ecosystems by introducing a cross-brand exchange powered by blockchain.

✨ Features
🔗 Multi-brand loyalty system
💰 Earn (mint) loyalty points
🔄 Swap points across brands (like forex exchange)
🧾 Redeem points
🔐 Role-based access (Admin & User)
🦊 MetaMask wallet integration
🌐 Live transaction verification via Etherscan
📱 Fully responsive UI
⏳ Auto point expiry system


🛠️ Tech Stack
Blockchain: Ethereum (Sepolia Testnet)
Smart Contract: Solidity
Frontend: HTML, CSS, JavaScript
Web3 Integration: ethers.js
Wallet: MetaMask
Deployment: Netlify


🏗️ Architecture
Frontend (HTML/CSS/JS + ethers.js)
           ↓
        MetaMask
           ↓
Smart Contract (Solidity)
           ↓
 Ethereum Sepolia Network
           ↓
    Etherscan (Verification)
⚙️ How It Works
1️⃣ Brand Registration

Brands register themselves on the platform.

2️⃣ Earn Points

Admin (brand) mints loyalty points to user wallets.

3️⃣ Swap Points

Users can swap points across brands based on predefined exchange rates.

4️⃣ Redeem Points

Users can redeem their loyalty points anytime.

🔐 Roles
Admin (Brand Owner):
Register brand
Mint points
Set exchange rates
User:
View balance
Swap points
Redeem points
